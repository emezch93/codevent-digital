Lesson 1: HTML5 Basics

What is HTML5?

HTML5 is the latest version of Hypertext Markup Language, the code that describes web pages.

Basic Structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First HTML5 Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML5 web page.</p>
  </body>
</html>
Lesson 2: Common HTML5 Elements

Essential Elements

  • <header> - Defines the header section
  • <nav> - Navigation links
  • <main> - Main content area
  • <footer> - Footer section
  • <section> - Logical sections of the page
