GitHub Pages Deployment Guide

Step-by-step instructions to deploy your CodeVent Digital React website on GitHub Pages for free

1

Create a GitHub Repository

First, create a new repository on GitHub to host your CodeVent Digital project.

# Go to https://github.com/new # Repository name: my-reponame # Make sure it's public (GitHub Pages only works with public repos for free accounts) # Click "Create repository"

After creating the repository, you'll see the repository URL: https://github.com/your-username/codevent-digital.git

2

Push Your React Project

Initialize your local React project as a Git repository and push it to GitHub.

# In your project root directory git init git add . git commit -m "Initial commit: my-mini-project" # Add remote repository (replace with your actual URL) git remote add origin https://github.com/your-username/my-mini-project # Push to GitHub git branch -M main git push -u origin main
3

Install gh-pages and Configure package.json

Install the gh-pages package and configure your package.json file to enable GitHub Pages deployment.

# Install gh-pages as a dev dependency npm install --save-dev gh-pages

Then, add these lines to your package.json file:

{ "name": "codevent-digital", "version": "0.1.0", "homepage": "https://your-username.github.io/my-mini-project", "dependencies": { // ... your existing dependencies }, "scripts": { "predeploy": "npm run build", "deploy": "gh-pages -d build", // ... your existing scripts } }
Important: Replace your-username with your actual GitHub username in the homepage field.
4

Run the Deploy Script

Execute the deploy script to build your React app and publish it to GitHub Pages.

# Run the deploy script npm run deploy

This command will:

  • Run npm run build to create an optimized production build
  • Deploy the contents of the build folder to the gh-pages branch
  • Push the changes to GitHub automatically

You should see output similar to this:

> my-mini-project@0.1.0 predeploy > npm run build > my-mini-project@0.1.0 build > react-scripts build Creating an optimized production build... Compiled successfully. > codevent-digital@0.1.0 deploy > gh-pages -d build Published
5

Access Your Live Website

Your website is now live! You can access it at the following URL:

https://your-username.github.io/codevent-digital

It may take a few minutes for GitHub Pages to process your deployment. If you see a 404 error initially, wait a few minutes and refresh.

Pro Tip: You can customize your GitHub Pages settings by going to your repository → Settings → Pages. Here you can set a custom domain, choose the source branch, and view deployment status.
