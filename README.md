# CodeVent Digital

> **Building Developers One Step at a Time**

A web development education platform and digital product marketplace for beginners — built with vanilla HTML, CSS, and JavaScript, hosted on GitHub Pages.

## 🔗 Live Site
[https://emezch93.github.io/codevent-digital/](https://emezch93.github.io/codevent-digital/)

## 📱 Progressive Web App
This site is a fully installable PWA. On Android Chrome, tap **"Add to Home Screen"** to install it like a native app.

## 📄 Pages
| Page | Description |
|---|---|
| `index.html` | Homepage — hero, journey pipeline, testimonials |
| `learning.html` | HTML/CSS/JS lessons, live editor, quiz |
| `community.html` | WhatsApp community funnel |
| `testimonial.html` | Success stories and student projects |
| `shop.html` | Digital marketplace — eBooks, courses, source code |
| `Contact.html` | Contact form (sends via WhatsApp) |
| `thankyou.html` | Course access page |

## 🛠 Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Firebase Authentication (Email/Password)
- GitHub Pages (hosting)
- Font Awesome 6 (icons)
- Google Fonts — DM Serif Display, Plus Jakarta Sans, DM Mono
