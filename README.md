# CodeVent Digital

> Building Developers One Step at a Time

**Live site:** https://codeventdigital.site

CodeVent Digital is a frontend developer education platform founded by
Ezenwa Chidera Emmanuel in November 2025, based in Surulere, Lagos,
Nigeria. It combines a structured web development curriculum, an
AI-assisted learning chat, an interactive developer toolkit, and a
catalog of digital products (eBooks, mini-courses) sold via Paystack.

## Stack

HTML · CSS · Vanilla JavaScript · Firebase Auth · Cloudflare Worker (AI chat, Gemini 2.5 Flash) · Cloudflare Pages

## PWA

Installable as a Progressive Web App — Chrome menu → Add to Home Screen.
Includes offline support (`sw.js`, `offline.html`) and a web app manifest
(`manifest.json`).

## SEO

- `sitemap.xml` — full page index for search engines
- `robots.txt` — crawler rules, points to the sitemap
